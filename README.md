# 🧠 Materi yang Dipelajari

## 1️⃣ List
List adalah struktur data yang digunakan untuk menyimpan banyak data dalam satu variabel.

Ciri-ciri:
- Bersifat **mutable**
- Mendukung **indexing dan slicing**
- Dapat menyimpan berbagai tipe data

Contoh:

```python
myList = [10, "hello", 14.5, False]
print(myList)
````

---

## 2️⃣ Tuple

Tuple adalah struktur data yang mirip dengan list tetapi **tidak dapat diubah (immutable)**.

Contoh:

```python
myTuple = (10, "hello", 14.5, False)
```

---

## 3️⃣ Set

Set adalah struktur data yang **tidak menyimpan data duplikat**.

Contoh:

```python
mySet = {1, 2, 3, 3, 4}
print(mySet)
```

---

## 4️⃣ Dictionary

Dictionary menyimpan data dalam bentuk **key-value pair**.

Contoh:

```python
myDictionary = {
    "nama": "John",
    "umur": 30
}
```
